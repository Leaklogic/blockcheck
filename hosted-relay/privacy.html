Mekworks eBay Connector

This page describes how the Mekworks eBay Connector handles information when used to access eBay APIs.

What This Integration Does

The Mekworks eBay Connector is used to authenticate with eBay and send API requests for browsing, account, inventory, and related eBay developer workflows.

Information We Process

  • eBay OAuth tokens returned during sign-in.
  • API request and response data needed to complete user actions.
  • Basic operational data required to troubleshoot integration issues.

How Information Is Used

  • To authenticate a user with eBay.
  • To make API requests explicitly requested by the user.
  • To display results inside the local workbench application.

Storage

In the current test setup, tokens are primarily held for active session use. Long-term storage of user data is not part of this relay site.

Third Parties

This integration communicates with eBay services as part of the normal OAuth and API request flow.
